# Megamania Remake - Atari Classic

Um remake fiel do clássico Megamania do Atari, desenvolvido com HTML5 Canvas e TypeScript.

## 🚀 Tecnologias Utilizadas

- **Vite**: Build tool e servidor de desenvolvimento.
- **TypeScript**: Tipagem estática para maior segurança no código.
- **HTML5 Canvas**: Renderização de gráficos 2D.
- **Vanilla CSS**: Estilização da interface e efeitos CRT.

## 🎮 Como Jogar

1. Clone o repositório:
   ```bash
   git clone https://github.com/maryreginaf15/jogoremake-megamania.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Abra o navegador no endereço indicado (geralmente `http://localhost:5173`).

### Controles
- **Setas Esquerda/Direita**: Movimentação.
- **Espaço**: Disparar laser.
- **Mobile**: Controles na tela disponíveis.

## 🛠️ Comandos de Build

- `npm run dev`: Inicia o servidor de desenvolvimento.
- `npm run build`: Gera os arquivos de produção na pasta `dist/`.
- `npm run preview`: Visualiza o build de produção localmente.
